Редакции DMDE (DMDE Editions)

Актуальная информация (сравнение редакций, покупка) - на сайте программы.

Free Edition содержит все основные функции, восстанавливает до 4000 файлов только из текущей панели.

В платных редакциях нет органичения при восстановлении данных.

Professional Edition имеет дополнительные функции и расширенные права для оказания услуг по восстановлению данных:

  • Портативный запуск без привязки к компьютеру
  • 64-разрядные версии (доступно более 2 ГБ RAM, только Windows и Linux)
  • Создание отчетов восстановления данных (включая логи и контрольные суммы)
  • Использование логов для продолжения посекторного копирования после остановки
  • Возможность настроить свой скрипт обработки ввода-вывода
  • Восстановление дополнительных потоков данных NTFS
  • DMA-доступ в DOS (для интерфейса ATA)
  • Одноразовая активация клиентских версий
