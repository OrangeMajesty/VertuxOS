Виртуальная реконструкция файловой системы

Откройте виртуальную папку [Все найденные + реконструкция] в панели файлов для виртуальной реконструкции структуры директорий и подготовки найденных и удаленных файлов и директорий для восстановления.

Могут использоваться реузльтаты Полного сканирования.

Для улучшения результатов можно попробовать изменить Параметры по умолчанию.

NTFS, extFS

"Чистая" реконструкция ФС

Наиболее точная реконструкция при отсутствии повреждений ФС.

Реконструкция "по умолчанию", меньше/больше результатов

Использование результатов Полного сканирования. Можно уменьшить/увеличить число результатов реконструкции. Кнопка MFT/Параметры позволяет вручную выбрать фрагменты для реконструкции.

Номера MFT

Фильтрация файлов по номерам для частичной реконструкции (в случае если полная реконструкция невозможна из-за недостатка памяти).

Cмещенные

Учет смещенных записей (необходимо Полное сканирование).

Найденные дополнительно

Включение дополнительно найденных файлов (возможно наличие мусора, однако, могут найтись некоторые потерянные файлы).

Тех. записи

Включение в результаты специальных записей для анализа специалистами NTFS.

Опции выбора фрагментов ФС

Корректные

Проверенные фрагменты, принадлежащие выбранному тому.

Связанные

При наличии нескольких вариантов одного тома (см. Полное сканирование) - фрагменты, принадлежащие другим вариантам выбранного тома.

Неизвестные

Фрагменты, принадлежность которых не удалось проверить.

Сторонние

Фрагменты, принадлежащие другим томам.

Мелкие (найденные дополнительно)

Файлы, найденные дополнительно (см. выше).

Недопустимые

Фрагменты, несовместимые с выбранным томом по параметрам ФС.

FAT/exFAT

Использовать только результат поиска (ex)FAT

Использовать только результаты Полного сканирования без повторного сканирования тома.

Пересканировать весь том

Дополнительно просканировать том, если он не полностью просканирован при Полном сканировании.

Настройки использования таблиц FAT ("Параметры")

Таблицы FAT содержат цепочки кластеров для сборки фрагментированных файлов и директорий. Программа автоматически выбирает вариант использования таблиц на основе оценки. Однако, можно попробовать разные параметры для улучшения реконструкции и результатов восстановления данных в целом или отдельных файлов.

Можно выбрать, какая из таблиц будет использоваться (FAT1 или FAT2), полностью отключить использование таблиц (не использовать FAT-таблицы), если таблицы сильно повреждены, или проверять блоки (не использовать плохие секторы) для обхода неправильных цепочек, если таблицы повереждены частично.
