Содержание · О программе

Описание файлов программы

Исполняемые файлы

dmde.exe / dmde - программа

Файлы локализаций

*.tbl - таблицы кодовых страниц
*.lng - языковые файлы

Файлы документации

eula*.txt - файлы лицензионного соглашения (EULA)
versions.txt - информация об изменениях в программе
readme*.txt - файлы Readme
dmde*.chm - файлы справки в формате Windows CHM
man*/* - файлы справки в формате HTML

Другие файлы

dev9x.dll, dev32.dll - драйверы для работы в Windows 98/ME
dmde*.ini - файлы настроек программы
cwsdpmi.exe - DOS Extender для запуска 32-разрядных приложений (Copyright © 2010 CW Sandmann, не является частью программы DMDE)
